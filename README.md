# 탈모 진단 챗봇 [자라나모]
> 카카오톡을 통해 손쉽게 자신의 탈모 진행 단계를 진단받을 수 있는 챗봇 서비스  

- 사용자는 챗봇을 통해 두피 이미지를 보내면, 자신의 탈모 단계를 확인할 수 있다.
- 이에 추가로 10가지 설문(문진)을 완료하면 자신의 두피 상태에 맞는 제품을 추천받을 수 있다. 
- 탈모 관련 궁금증을 해결할 수 있는 QnA를 제공한다.
