# 흐름도

7가지 (단계 질문)

0단계일 경우 영양제/두피마사지/헤어토닉에 대한 유형 질문 한가지만 추가해서 물어봄-> 7+1 총 8개
3단계일 경우 흑채/부분가발에 대한 유형 질문 한가지만 물어봄 -> 7+1 총 8개
1,2단계일 경우 건성/지성 타입 질문 5가지, 민감성에 대한 질문 3가지, 피지 질문 1가지, 모발 굵기에 대한 질문 1가지-> 7+10 총 17개
